# Mermaider GitHub Action

Mermaider GitHub Action is an action used to generate mermaid diagrams from
code.

## Usage

```yaml
- uses: mermaider/mermaider@v1
  with:
    # The path to the file(s) to parse
    file: path/to/file.ts
```
