# Mermaider CLI

## Installation

```bash
npm install -g @mermaider/cli
# or
yarn global add @mermaider/cli
# or
pnpm add -g @mermaider/cli
```

## Usage

```bash
mermaider parse ./path/to/file.ts
```

This will generate a `.mmd` file and a `.svg` file in the same directory as the file.
